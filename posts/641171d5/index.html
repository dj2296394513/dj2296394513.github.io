# 异步提交表单

# 异步提交表单的步骤

  • 获取表单及所有表单组件对应的数据值。
  • 将所有表单组件对应的数据值拼成特定格式的字符串或是 JSON 格式数据。
  • 通过 Ajax 异步交互方式提交表单。

# 表单序列化

  • serialize () 方法：将表单组件对应的数据值序列化为指定格式的字符串内容。

  • serializeArray () 方法：将表单组件对应的数据值序列化为 JSON 格式的数据内容。
