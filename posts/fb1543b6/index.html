# 如何提高搜索效率？

# Google

# 1. 逻辑关系 (与 / 或)

谷歌默认的就是 与的逻辑搜索 ，进行搜索时，关键词都会出现在页面搜所结果中

关键字 OR 关键字 OR 关键字

# 2. 精确查找 (" ")

在关键词两边分别加一个 「引号("")」 , 能快速的进行关键词的匹配

"关键字"

# 3. 搜索的关键字包含在网页标题中 (intitle)

对关键词包含在 title 中的搜索

intitle : 搜索内容

# 4. 使用同义词 (~)

谷歌内置的 优化方法

~ 关键字

# 5. 使用限定文件格式 (filetype)

找一些 文件格式 的文件

关键字 filetype:doc OR filetype:xls OR filetype:ppt OR filetype:pdf

# 6. 使用减号 (非) 来进行搜索

搜索的网页中 不包含 一些关键词

搜索内容 - 关键字 - 关键字

# 7. 限定网站域名搜索 (site)

用于在某一个 网站 搜索相关内容

搜索内容 site : www.xxx.com
搜索内容 site : .com

# 百度 (baidu)

# 1.‘A -B’
  • 作用：搜索 A 屏蔽 B 的信息
  • 例如：科学 - 技术

- 前必须有空格

# 2.‘A | B’
  • 作用：搜索包含 A 的信息或者包含 B 的信息
  • 例如：学习 | 知识
# 3.‘A intitle:B’
  • 作用：把搜索范围限定在网页标题
  • 例如 : ps intitle: 技巧

‘intitle:’后面不能有空格

# 4.‘A site: 站点域名’
  • 作用：将搜索范围界定在指定网站中率
  • 例如：百度杀毒 site:www.skycn.com
# 5.‘: A inurl: 特定 url’
  • 作用：将搜索范围界定到 Url 链接中
  • 例如：帮助 inurl:baidu.com
# 6.‘A’《A》
  • 作用：将搜索范围更精确的指定到所要查的内容中，避免关键词被拆分
  • 例如 : “三国演义 " 《三国演义》不加书名号也可能是电视剧
# 7. A filetype: 文件类型
  • 作用：搜索该类型的文档资料
  • 例如：实习报告 file:doc

百度文档搜索界面 (http://file.baidu.com )

# 8. 电影名迅雷下载
  • 作用：用迅雷下载电影
  • 例如：三国演义迅雷下载
