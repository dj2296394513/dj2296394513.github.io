# 箭头函数

匿名函数的一个简写

简写规则:

  • a. function 改成 => = > 可以读成 goesto
  • b. 如果只有一个形参，那就可以省略形参小括号.
  • c. 如果不是一个形参，0 个或者多个形参， 那就不能省略这个形参小括号了
  • d. 如果函数体只有一句话，那就可以省略函数体的大括号
  • e. 如果函数体只有一句话，并且这一句话是 return 返回值，那 return 也要省略.
  • f. 如果函数体不是一句话， 那就不能省略这个大括号.
