# Vue 简介

# Vue.js 是什么

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架.

什么是渐进式？

  • 渐进式意味着你可以将 Vue 作为你应用的一部分嵌入其中，带来更丰富的交互体验。
  • 如果你希望将更多的业务逻辑使用 Vue 实现，那么 Vue 的核心库以及其生态系统

特点:

  • 解耦视图和数据
  • 可复用的组件
  • 前端路由技术
  • 状态管理
  • 虚拟 DOM

# 声明式渲染

  • v-bind attribute 被称为**指令**。指令带有前缀 v-`，以表示它们是 Vue 提供的特殊 attribute

# 条件与循环

  • v-if 指令判断内容是否显示
  • v-for 指令可以绑定数组的数据来渲染一个项目列表

如果添加一个内容化，则会直接加到已有内容的后面

# 处理用户输入

  • v-on 指令添加一个事件监听器

  • v-model 指令实现表单输入和应用状态之间的双向绑定。
