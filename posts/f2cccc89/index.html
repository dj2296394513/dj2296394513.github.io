# MySQL 库操作

  • 创建数据库
create database 数据库名称;
  • 删除数据库
drop database 数据库名称;

删除数据库是不可能找回的

  • 查看数据库

    • 查看所有数据库
    show databases;
    • 查看指定数据库
    show databases like 'db%'; //查询所有以db开头的所有数据库

    _(下划线)：代表任一一个字符，可能代表 a 也可能代表 b...

    %(百分号)：代表任意的 n 个字符，可能代表 ab 也可能代表 abcd...

  • 修改数据库

ALTER DATABASE [数据库名] { 
[ DEFAULT ] CHARACTER SET <字符集名> |
[ DEFAULT ] COLLATE <校对规则名>}

语法说明如下：
	ALTER DATABASE 用于更改数据库的全局特性。
	使用 ALTER DATABASE 需要获得数据库 ALTER 权限。
	数据库名称可以忽略，此时语句对应于默认数据库。
	CHARACTER SET 子句用于更改默认的数据库字符集。

# MySQL 数据表操作

  • 创建数据表
CREATE TABLE table_name (column_name column_type);
  • 删除数据表
drop table 数据表名称;
  • 修改数据表
rename table 原来的名字 to 新名称;

  • 查看数据表

    • 查看所有数据表或某类数据
    show tables like '_或%';
    • 查看数据表结构
    desc 数据表名称;

# MySQL 中数据的操作

  • 插入数据
insert into 数据表名称([字段1，字段2...]) values (字段值1，字段值2...)
  • 更新数据
update 数据表名称 set 字段(列) = 更新后的值,字段(列) = 更新后的值 where 条件;

  • 查询数据

    • select 语法

      • 查询所有数据
      SELECT * FROM 数据表名称
      • 查询单个数据
      SELECT column_name,column_name FROM 数据表名称

    • where 语法

    SELECT field1 FROM table_name1 WHERE `id`>5
    操作符描述实例
    =等号，检测两个值是否相等，如果相等返回 true(A=B) 返回 false
    <>,!=不等于，检测两个值是否相等，如果不相等返回 true(A!=B) 返回 true
    >大于号，检测左边的值是否大于右边的值，如果左边的值大于右边的值返回 true(A>B) 返回 false
    <小于号，检测左边的值是否小于右边的值，如果左边的值小于右边的值返回 true(A<B) 返回 true
    >=大于等于号，检测左边的值是否大于或等于右边的值，如果左边的值大于或等于右边的值返回 true(A>=B) 返回 false
    <=小于等于号，检测左边的值是否小于于或等于右边的值，如果左边的值小于或等于右边的值返回 true(A<=B) 返回 true
    • order by 语法
    -- ASC 升序
-- DESC 降序
SELECT field table_name ORDER BY `id` ASC
    • limit 语法
    -- 0 表示起始点 5表示显示条数 可以判定显示为第一页
SELECT * FROM `table_name` WHERE `id` LIMIT 0,5;

    # LIMIT 可以通过得出起始点，算法如下：

    （当前页数 - 1） * 显示的条数 = 起始点

  • 删除数据

delete from 数据表名称 where 条件;

删除表 : delete、drop、truncate 三者的区别:

  • delete 和 truncate 仅仅删除表数据，drop 连表数据和表结构一起删除，打个比方，delete 是单杀，truncate 是团灭，drop 是把电脑摔了。
  • delete 是 DML 语句，操作完以后如果没有不想提交事务还可以回滚，truncate 和 drop 是 DDL 语句，操作完马上生效，不能回滚，打个比方，delete 是发微信说分手，后悔还可以撤回，truncate 和 drop 是直接扇耳光说滚，不能反悔。
  • 执行的速度上，drop>truncate>delete，打个比方，drop 是神舟火箭，truncate 是和谐号动车，delete 是自行车。
