# axios 的请求合并以及参数配置

参考简书 1

参考简书 2

  • 并发请求 -- 处理并发请求的助手函数

    • axios.all (iterable) -- iterable 是一个可以迭代的参数如数组等
    • axios.spread (callback) -- callback 要等到所有请求都完成才会执行

  • 配置 config 来发送请求

    • axios(config)
    • axios(url[,config])

  • 请求方式的别名

  • 初始化常用配置

    • baseURL -- 默认地址

    • validateStatus -- 状态码，回复状态 resolve 或者 reject promise

      • 默认状态码
      • 自定义成功失败规则 -- resolve/reject

    • timeout -- 延迟时间

    • headers -- 设置请求头信息
    • params -- GET 传参
    • data -- POST 传参
    • 支持格式 -- 默认 RAW -- 常用 URL-ENCODEED
    • common -- 当实例创建时候修改配置
    • 设置响应拦截器 -- 分别在响应成功和失败的时候做一些拦截处理（在执行成功后设定的方法之前，先会执行拦截器中的方法）
    • transformRequest -- 允许在请求数据发送到服务器之前对其进行更改
