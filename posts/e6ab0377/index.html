# 显示和隐藏动画

show () -- 显示

hide () -- 隐藏

# 展开和收入动画

slideDown () -- 展开

slideUp () -- 收缩

# 淡入淡出动画

fadeIn () -- 淡入

fadeOut () -- 淡出

fadeTo -- 淡入到

# 自定义动画

animate () -- 动画

  • 第一个参数：接收一个对象，可以在对象中修改属性
  • 第二个参数：指定动画时长
  • 第三个参数：指定动画节奏，默认就是 swing
  • 第四个参数：动画执行完毕之后的回调函数

# stop 和 delay 方法

stop () -- 停止

delay () -- 延迟

# stop () 方法

  • 立即停止当前动画，继续执行后续的动画

  • 立即停止当前和后续所有的动画

  • 立即完成当前的，继续执行后续动画

  • 立即完成当前的，并且停止后续所有的

# delay () 方法

用于告诉系统延迟时长
