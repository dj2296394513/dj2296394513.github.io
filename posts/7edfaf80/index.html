# 模板语法

# 文本
  • Mustache 语法

(双大括号) 的文本插值

无论何时，绑定的数据对象上 msg property 发生了改变，插值处的内容都会更新。

  • v-once 指令

该指令后面不需要跟任何表达式 (比如之前的 v-for 后面是由跟表达式的)

该指令表示元素和组件 (组件后面才会学习) 只渲染一次，不会随着数据的改变而改变

# 原始 HTML
  • v-html 指令

该指令后面往往会跟上一个 string 类型

会将 string 的 html 解析出来并且进行渲染

不要对用户提供内容使用插值

# Attribute
  • v-bind 指令

作用：动态绑定属性
缩写：:
预期：any (with argument) | Object (without argument)
参数：attrOrProp (optional)

# 使用 JavaScript 表达式

模板表达式都被放在沙盒中，只能访问全局变量的一个白名单，如 Math 和 Date 。你不应该在模板表达式中试图访问用户定义的全局变量。

# 指令

带有 v- 前缀的特殊 attribute

预期是单个 JavaScript 表达式 ( v-for 除外)

# 参数

在指令名称之后以冒号表示

# 动态参数

用 [ ]

动态参数预期会求出一个字符串，异常情况下值为 null

# 修饰符

以半角句号 . 指明的特殊后缀，用于指出一个指令应该以特殊方式绑定
